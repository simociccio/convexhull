Descrizione dettagliata

template<class T>
class Point< T >

La classe Point modella un punto o un vettore nell spazio.

Rappresenta un punto o vettore in 3 dimensioni, con precisione dati variabile definita dal tipo T. In particolare, è possibile avere punti/vettori con precisione intera, float o double.

§ operator<()

template<class T>
bool Point< T >::operator< ( const Point< T > &  otherPoint) const
inline

Operatore di minore tra punti/vettori.

In questo contesto, il punto/vettore è minore di otherPoint se la sua componente x è minore di quella di otherPoint; in caso di uguaglianza si verifica la componente y e in caso di ultieriore uguaglianza la componente z.

Parametri
[in]otherPointaltro punto/vettore
Restituisce
True se il punto/vettore this è minore di otherPoint, false altrimenti
